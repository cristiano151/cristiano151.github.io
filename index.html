Gestão de Oficina Auto

Aplicação local que permite gerir uma oficina de mecânica de automóveis.

Este foi o meu Projeto de Aptidão Profissional, que apresentei no final do meu Curso de Secundário (Técnico de Informática de Gestão), foi desenvolvida com Linguagem de Programação Java e possui ligação a uma base de dados SQL. Esta aplicação, possui vários módulos que são essenciais à gestão de uma oficina, tais como:

  • Gestão de clientes (marcações, veículos, recebimentos);
  • Gestão de veículos de clientes;
  • Gestão de serviços prestados (checklist, orçamentos e reparação);
  • Gestão de artigos e respetivos valores em stock;
  • Gestão de fornecedores
  • Gestão de recebimentos e pagamentos
  • Gestão de terceiros
  • Gestão de marcações
